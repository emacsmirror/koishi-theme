#+title: koishi-theme

The Emacs theme in your subconsciousness!

* About

A simple color theme based on green-yellow-purple colors, support both light and dark theme.
This is the official theme of [[https://github.com/gynamics/koishimacs][Koishimacs]] (my Emacs config).
Inspired by Touhou Project's character Komeiji Koishi's color tone.

I try to make it in a colorful but undersaturated style.
But the output is oversaturated.

* Installation
You can install ~koishi-theme~ from MELPA, this is an example with ~use-package~.
#+begin_src emacs-lisp
  (use-package koishi-theme
    :ensure t
    :init
    ;; set `koishi-theme-mode' to t if you want to use light theme,
    ;; this variable should be configured before you load theme.
    (setq koishi-theme-mode t)
    ;; enable koishi theme
    (load-theme 'koishi))
#+end_src

From version 3.0 we added some experimental features to all the user modify the behavior of ~koishi-theme~ after it is defined and loaded. The user can put some filter functions (called 'effects') to ~koishi-theme-effects~, and then call ~koishi-theme-reset-faces~ to see the changes. Currently we have implemented some configurable effects, you may configure their behavior with related custom variables.

- ~koishi-theme-effect--complement~: Use complementary color. The light theme is generated by this effect.
- ~koishi-theme-effect--afterglow~: Blending with a give shader color.
- ~koishi-theme-effect--hsl-bias~: Add random bias in HSL color space.

You can find related custom variables by calling ~M-x customize-group RET koishi RET~.
